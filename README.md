## 📖 About

This is my **professional portfolio website** developed during my Web Development Internship at **Arch Technologies**. The portfolio showcases my projects, technical skills, and professional experience in a modern, dark-themed, fully responsive interface.

🔗 **Live Demo:** [zohaibullah19.github.io/portfolio](https://zohaibullah19.github.io/portfolio/)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **Modern Dark Theme** | Sleek charcoal background with electric blue accents |
| 📱 **Fully Responsive** | Optimized for desktop, tablet, and mobile devices |
| 🧭 **Fixed Navigation** | Smooth scrolling navigation with active section highlighting |
| 👤 **Professional Profile** | Circular photo with hover glow and scale effects |
| 🛠️ **Skills Grid** | Interactive badges displaying technical competencies |
| 📅 **Experience Timeline** | Chronological work history with detailed descriptions |
| 🖼️ **Projects Showcase** | CSS Grid layout with hover zoom effects on project images |
| 🏷️ **Tech Stack Badges** | Visual tags showing technologies used in each project |
| 🔗 **GitHub Integration** | Direct repository links for all projects |
| 📧 **Contact Section** | Functional contact form with LinkedIn and GitHub links |
